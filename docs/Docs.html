M:CTree.Factory.RepeatWhile(CTree.World,System.String)

Executes all sub-behaviors in parallel until the condition because false. The condition is re-evaluated before each iteration. Each iteration ends when all sub-behaviors complete.

M:CTree.Factory.If(CTree.World,System.String)

Executes all sub-behaviors in parallel if the condition is true. Otherwise, completes immediately. A condition must be based on a stored value. C# code

    If the variable, `Solved`, has value 1, execute the sub-behaviors
    
    If: Solved, 1
      DoStuff
    End
M:CTree.Factory.IfClick(CTree.World,System.String)

Executes all sub-behaviors in parallel if the condition is true. Otherwise, completes immediately. The condition triggers if the user clicks the given scene object. C# code

    Click on the asset named `SceneObjectName`
  
  IfClick: SceneObjectName
     DoStuff
  End
