Remarks

Use this Behavior to implement animations and other effects that occur over multiple frames. Configure CoroutineBehavior with a function that takes World as input and returns an iterator. For example, the following code increases the size of xform to twice its original size over 3.0 seconds.

Transform xform = world.Get(objName.Trim());
Behavior b = new CoroutineBehavior(world, 
     ProceduralAnimator.Grow(xform, 1, 2, 3.0));

Coroutines cannot be anonymous functions. Our Coroutines for animation behaviors are implemented in ProceduralAnimator.
